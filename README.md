# Nyanko Strike Rivals

猫耳ボウラーの Mao がライバルの Rinka と対戦する、ブラウザ向け 3D ボウリングゲームです。キャラクター演出はピクセルアート風の 2D スプライトで表示し、レーン、ボール、ピンの挙動は Three.js と Rapier による 3D 物理シミュレーションで動かしています。

## 操作方法

- `A` / `D`: 投球位置を左右に移動
- `←` / `→`: 投球角度を調整
- `Space` または `Start Power` ボタン: タイミング入力を進める
- 1 回目のタイミング: ボール速度を決定
- 2 回目のタイミング: カーブの強さを決定
- `Reset`: 現在の勝負をリセット

ゲームは 5 フレーム制です。ストライク、スペア、ガーターなどの結果に応じて、中央上部の演出ウィンドウとピン奥のサイネージが変化します。

## クイックスタート

```bash
npm install
npm run dev
```

ブラウザで表示されたローカル URL を開いてください。通常は次の URL です。

```text
http://127.0.0.1:5173/
```

本番ビルドを確認する場合:

```bash
npm run build
npm run preview
```

## GitHub Pages

`vite.config.ts` の `base` は `./` にしているため、GitHub Pages のサブパス配信でもアセットを相対パスで解決できます。

公開用の静的ファイルは次のコマンドで `dist/` に生成されます。

```bash
npm run build
```

## 使っている技術

- TypeScript: ゲームロジックと型定義
- Vite: 開発サーバーと本番ビルド
- Three.js: 3D レーン、ボール、ピン、カメラ、ライティング
- Rapier 3D: ボールとピンの物理挙動
- DOM / CSS: HUD、スコア、操作パネル、演出ウィンドウ
- 生成画像アセット: キャラクター立ち絵、スプライト、勝敗一枚絵

## 制作時の依頼概要

このゲームは、ユーザーから次のような依頼を受けて制作しました。

- 参考三面図のキャラクターをもとに、ボウリングウェアのヒロインを作る
- ヒロインとライバルがボウリングで対戦するブラウザゲームにする
- キャラクターはイラストやピクセルアート風アニメーションで表現する
- ボウリング部分は実規格に近いサイズの 3D 物理挙動で表現する
- 速度とカーブをタイミング入力で決めるゲーム性を入れる
- スコア、セリフ、サイネージ、勝敗一枚絵などの演出を追加する
- GitHub Pages で公開しやすい構成にする
